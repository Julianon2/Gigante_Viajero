## Frameworks y Librerías

### Backend
- **Express**: Framework principal para API REST.
- **Mongoose**: ODM para MongoDB.
- **MongoDB**: Base de datos NoSQL.
- **Firebase / firebase-admin**: Integración para autenticación y servicios cloud.
- **Stripe**: Pagos.
- **Nodemailer**: Envío de emails.
- **OpenAI**: Integración IA.
- **Helmet, xss-clean, express-mongo-sanitize**: Seguridad.
- **jsonwebtoken**: Autenticación JWT.
- **bcrypt/bcryptjs**: Hashing de contraseñas.
- **express-rate-limit**: Rate limiting.
- **multer**: Uploads.
- **winston**: Logging.
- **morgan**: Logging HTTP.
- **dotenv**: Variables de entorno.
- **cors**: CORS.
- **express-validator**: Validación.

### Frontend
- **No frameworks detectados** (HTML, CSS, JS puro)
